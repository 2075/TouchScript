What happens when you add or remove a point from cluster:
  • Nothing — Nothing happened. Point hasn't been added.
  • PointAdded — Point successfully added.
  • FirstPointAdded — That was the first point added to cluster.
  • PointRemoved — Point successfully removed.
  • LastPointRemoved — That was the last point removed from cluster.
