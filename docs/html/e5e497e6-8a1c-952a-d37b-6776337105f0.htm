The FlickGesture type exposes the following members.
Fields
  NameDescription
activeTouches (Inherited from Gesture.)
FlickTime
Time interval in seconds in which points cluster must move by MinDistance.
Horizontal
If true, tracks only horizontal movement
MinDistance
Minimum distance in cm to move in FlickTime before ending gesture for it to be recognized.
MovementThreshold
Minimum distance in cm for cluster to move to be considered as a possible gesture. Prevents misinterpreting taps.
Vertical
If true, tracks only vertical movement
WillRecognizeWith
An array of gestures which are added to ShouldRecognizeSimultaneouslyWith(Gesture, Boolean) DURING STARTUP so these other gestures could work with this one.
